# eBooks Library

A modern web application for browsing and reading books from the Gutenberg Project. Built with React, TypeScript, and Vite, this app provides an intuitive interface to explore thousands of free eBooks across multiple categories.

## Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - CSS framework
- **Gutendex API** - Book data source

## Getting Started

- Node.js (v18 or higher recommended)
- npm

## Installation

1. Clone the repository:
git clone <repo-url>
cd eBooks-library

2. Install dependencies:
npm install

3. Start the development server:
npm run dev

4. Build for Production:
npm run build


### Browsing Books(Usecase)
1. **Select a Category**: On the home page, click on any category card to view books in that category
2. **Search**: Use the search bar to find books by title or author name
3. **View Book**: Click on any book card to open it in a new tab (if a viewable format is available)
4. **Load More**: Scroll down to automatically load more books
